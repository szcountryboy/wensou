# wensou

#### 介绍
windows平台暂不支持pdf、office，对常规的文件内容搜索， 性能较低。基于此，实现本地文件内容搜索。

支持类型：常规文本、pdf、office.

后续会支持图片、压缩文件、音频、视频、自定义格式
